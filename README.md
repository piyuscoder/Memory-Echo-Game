## 🕹️ How to Play

1. **Start the Game**  
   - Desktop: Press any key  
   - Mobile: Tap anywhere  

2. **Watch the Sequence**  
   - Colors will flash in a specific order  

3. **Repeat the Sequence**  
   - Click/tap the buttons in the same order  

4. **Level Up**  
   - Every correct round adds a new color to the sequence
